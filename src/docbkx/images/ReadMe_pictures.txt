SVG files are extracted from the following document:
https://teamforge.schneider-electric.com/sf/go/doc30171

Steps to transform a PPT slide to a SVG :
 - In PowerPoint, save the slide as a new file and select EMF (Enhanced Windows MetaFile) format
 - Open the EMF file with Inkscape an perform the following actions :
  + Remove the unwanted parts of the SVG content
  + Select File -> Document properties..., then Resize page to content..., then click on Resize page to drawing or selection
  + Save file as Plain SVG
  
Tips:
If facing issue with fonts, use standard fonts: 
 - Sans or sans-serif
 - serif
 - monotype
